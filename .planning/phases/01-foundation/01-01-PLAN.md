  <name>Task 2: Create app shell with Corbusier theme, Zustand routing, and placeholder screens</name>
  <files>src/index.css, src/store/appSlice.ts, src/App.tsx, src/main.tsx, src/screens/ConfigScreen.tsx, src/screens/GameScreen.tsx, index.html</files>
  <action>
1. Create `src/index.css` with Tailwind v4 theme (NO tailwind.config.js — v4 uses CSS-first config):
   ```css
   @import "tailwindcss";
   @import url('https://fonts.googleapis.com/css2?family=Jost:wght@400;700&display=swap');

   @theme {
     --color-corbusier-red: #d0021b;
     --color-corbusier-blue: #003f91;
     --color-corbusier-yellow: #f5a623;
     --color-concrete: #f2f0eb;
     --color-charcoal: #3a3a3a;
     --font-jost: 'Jost', sans-serif;
   }
   ```
   This auto-generates utility classes: `bg-corbusier-red`, `text-corbusier-blue`, `font-jost`, etc.

2. Create `src/store/appSlice.ts` — Zustand v5 store for screen navigation:
   ```typescript
   import { create } from 'zustand'

   type Screen = 'config' | 'game'

   interface AppState {
     screen: Screen
     setScreen: (screen: Screen) => void
   }

   export const useAppStore = create<AppState>()((set) => ({
     screen: 'config',
     setScreen: (screen) => set({ screen }),
   }))
   ```

3. Create `src/screens/ConfigScreen.tsx` — placeholder with Le Corbusier styling:
   - Display "Word Chicken" as heading using `font-jost font-bold uppercase tracking-widest text-charcoal`
   - Display a "Start Game" button styled with `bg-corbusier-red text-white font-jost font-bold uppercase px-8 py-3`
   - Sharp corners (no rounded), no shadows — per user decision
   - Button calls `useAppStore(s => s.setScreen)('game')` on click

4. Create `src/screens/GameScreen.tsx` — placeholder with Le Corbusier styling:
   - Display "Game Screen" heading with same Jost typography
   - Display a "Back to Config" button styled with `bg-corbusier-blue text-white` same sharp style
   - Button calls `useAppStore(s => s.setScreen)('config')` on click

5. Update `src/App.tsx` — root component:
   - Import `useAppStore` from store
   - Read `screen` from store
   - Conditionally render `ConfigScreen` or `GameScreen`
   - Wrap in a container div with `bg-concrete min-h-screen font-jost`

6. Update `src/main.tsx` to import `./index.css` and render `<App />` (standard Vite pattern).

7. Update `index.html` title to "Word Chicken".

Visual direction per user decisions: Light mode only, off-white/concrete gray background, bold primary color accents (red, blue, yellow), sharp corners, no shadows, Jost typography with bold uppercase headings and letter-spacing.
