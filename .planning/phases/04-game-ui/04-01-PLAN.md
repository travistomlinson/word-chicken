Create `src/components/TileCard.tsx` -- a reusable tile button component used throughout the game.

Props interface:
```typescript
interface TileCardProps {
  letter: string          // Single char; 'Q' renders as 'Qu'
  size?: 'sm' | 'md' | 'lg'
  onClick?: () => void
  color?: 'red' | 'blue' | 'yellow' | 'concrete'
  disabled?: boolean
  className?: string      // Allow additional classes (for animation)
}
```

Implementation:
- Display: `letter === 'Q' ? 'Qu' : letter`
- Size classes: sm = `w-8 h-8 text-sm`, md = `w-10 h-10 text-base`, lg = `w-12 h-12 text-lg`
- Color classes mapped to Corbusier theme: red = `bg-corbusier-red text-white`, blue = `bg-corbusier-blue text-white`, yellow = `bg-corbusier-yellow text-white`, concrete = `bg-concrete text-charcoal border border-charcoal/20`
- Render as `<button>` with `onClick` and `disabled` props
- Base classes: `font-jost font-bold uppercase flex items-center justify-center rounded select-none`
- When disabled: `opacity-50 cursor-not-allowed`
- Min touch target: always at least 44x44px on mobile via `min-w-[44px] min-h-[44px]`

Add to `src/index.css` (after the existing @theme block):
```css
@keyframes shake {
  0%, 100% { transform: translateX(0); }
  20%      { transform: translateX(-6px); }
  40%      { transform: translateX(6px); }
  60%      { transform: translateX(-4px); }
  80%      { transform: translateX(4px); }
}

.animate-shake {
  animation: shake 0.4s ease-in-out;
}
```

**HowToPlayModal** (`src/components/HowToPlayModal.tsx`):

Props: `{ isOpen: boolean; onClose: () => void }`

Structure:
- When `isOpen` is false, render nothing
- Fixed overlay: `fixed inset-0 z-50 flex items-center justify-center bg-black/50`
- Inner card: `bg-white max-w-md w-full mx-4 p-6 rounded`
- Title: "How to Play" with `font-jost font-bold uppercase tracking-widest text-2xl mb-4`
- Rules content (hardcoded text):
  1. Each round starts with a 3-letter word
  2. Take turns adding one letter to grow the word
  3. Rearranging letters is allowed (CAT + R = CART)
  4. If you can't make a word, you're eliminated
  5. Last player standing wins the round
  6. Q counts as "Qu" and uses one tile
- "Got It" dismiss button: `bg-corbusier-blue text-white font-jost font-bold uppercase px-6 py-2 rounded`

**ConfigScreen** (`src/screens/ConfigScreen.tsx`):

Replace the placeholder entirely. Implementation:

1. **localStorage persistence:**
   - Storage key: `'word-chicken-config'`
   - Stored shape: `{ difficulty: 'easy'|'medium'|'hard', banPluralS: boolean, tileDistribution: 'bananagrams'|'scrabble' }`
   - Defaults: `{ difficulty: 'medium', banPluralS: true, tileDistribution: 'bananagrams' }`
   - On mount: `useState` initialized from `loadConfig()` which reads localStorage with try/catch fallback to defaults
   - On any config change: immediately write to localStorage via `saveConfig()`

2. **Layout:**
   - Centered column layout, `min-h-screen bg-concrete p-4`
   - Title: "Word Chicken" (existing style: `font-jost font-bold uppercase tracking-widest text-charcoal text-4xl sm:text-5xl mb-8`)
   - "How to Play" link below title: `text-corbusier-blue underline cursor-pointer text-sm font-jost uppercase`

3. **Difficulty selector:**
   - Label: "Difficulty" in `font-jost font-bold uppercase tracking-wider text-charcoal text-sm mb-3`
   - Three stacked cards (not side-by-side):
     - Easy: `bg-corbusier-blue` background, white text, "Easy" title + "Common words" subtitle
     - Medium: `bg-corbusier-yellow` background, white text, "Medium" title + "Broader vocabulary" subtitle
     - Hard: `bg-corbusier-red` background, white text, "Hard" title + "Full dictionary" subtitle
   - Selected card: `ring-2 ring-charcoal ring-offset-2` (ring to indicate selection)
   - Unselected: `opacity-60`
   - Cards are buttons with `w-full p-4 rounded mb-2 text-left`

4. **Rules section:**
   - Label: "Rules" in same style as Difficulty label, with `mt-6 mb-3`
   - Plurals toggle: A row with label "Allow plurals (adding S)" and a toggle switch
     - Toggle: a `<button>` with `w-12 h-6 rounded-full` background that is `bg-corbusier-blue` when ON, `bg-charcoal/30` when OFF
     - Inner circle: `w-5 h-5 rounded-full bg-white` that translates right when ON
     - Default: OFF (banPluralS = true means plurals are banned, toggle OFF = ban)
   - Tile distribution: Two small buttons side by side
     - "Bananagrams" and "Scrabble" -- selected has `bg-charcoal text-white`, unselected has `bg-white text-charcoal border border-charcoal/20`

5. **Start Game button:**
   - Full width, `bg-corbusier-red text-white font-jost font-bold uppercase text-lg py-3 rounded mt-8`
   - onClick handler:
     ```typescript
     const dictionary = useDictionaryStore.getState().words
     const dispatch = useGameStore(s => s.dispatch)
     dispatch({
       type: 'START_GAME',
       config: { difficulty, banPluralS, tileDistribution, dictionary }
     })
     setScreen('game')
     ```

6. **Max width:** Entire content wrapped in `max-w-md mx-auto` for readable column width

IMPORTANT: The `banPluralS` state logic -- the toggle label says "Allow plurals" so when toggle is ON, `banPluralS = false`. When toggle is OFF (default), `banPluralS = true`.
