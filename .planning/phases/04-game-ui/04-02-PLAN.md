Props: `{ wordLength: number }`

Per CONTEXT.md locked decision: vertical thermometer, right side, blue->yellow->red gradient, purely abstract, 300ms transition.

Implementation (overlay-mask approach from RESEARCH):
- Outer container: `relative h-64 w-4 rounded overflow-hidden` with full gradient background via inline style: `background: linear-gradient(to top, #003f91, #f5a623, #d0021b)`
- Inner mask: `absolute top-0 left-0 right-0 bg-concrete transition-all duration-300 ease-out` with `style={{ height: \`${100 - tensionPercent(wordLength)}%\` }}`
- `tensionPercent(len)`: `Math.min(100, (len / 15) * 100)` -- 15 letters = full bar
- No labels, no numbers, no icons (locked decision)
