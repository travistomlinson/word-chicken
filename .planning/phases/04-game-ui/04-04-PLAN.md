From src/types/game.ts:
```typescript
export interface GameState {
  phase: TurnPhase
  round: RoundState
  config: GameConfig
  roundScores: Record<string, number>   // Points earned THIS round
  totalScores: Record<string, number>   // Cumulative across all rounds
}

export interface RoundState {
  roundNumber: number
  currentWord: string
  currentPlayerId: string
  activePlayers: string[]
  players: Record<string, PlayerState>
  bag: string[]
  turnHistory: Array<{ playerId: string; word: string; score: number }>
}

export type GameAction =
  | { type: 'START_GAME'; config: GameConfig }
  | { type: 'END_ROUND' }
  | { type: 'NEXT_ROUND'; winnerId: string }
  | { type: 'RESET_GAME' }
```

Key reducer behaviors:
- END_ROUND: Computes roundScores from turnHistory, accumulates into totalScores. Only valid during ROUND_END phase.
- NEXT_ROUND: Creates new round with winnerId as starting player. Resets roundScores. Returns to SETUP phase.
- RESET_GAME: Returns null (store clears gameState).
- START_GAME: Creates fresh GameState from config.

**Tile scatter animation in PlayerHand:**

Per CONTEXT.md locked decision: "When a player is eliminated, their hand tiles scatter/fall off screen with animation"

Add to PlayerHand:
```typescript
const [eliminating, setEliminating] = useState(false)
const wasActive = useRef(true)

const humanPlayer = useGameStore(s => s.gameState?.round.players['human'])
const isActive = humanPlayer?.isActive ?? true

useEffect(() => {
  if (wasActive.current && !isActive) {
    setEliminating(true)
    // Reset after animation completes
    const timer = setTimeout(() => setEliminating(false), 800)
    return () => clearTimeout(timer)
  }
  wasActive.current = isActive
}, [isActive])
```

When `eliminating` is true, each hand tile gets a random scatter transform via inline style:
```typescript
function scatterStyle(index: number): React.CSSProperties {
  if (!eliminating) return {}
  const angle = (index * 45 + Math.random() * 30) - 90  // spread in arc
  const distance = 200 + Math.random() * 300
  const x = Math.cos(angle * Math.PI / 180) * distance
  const y = Math.sin(angle * Math.PI / 180) * distance - 200  // bias upward then fall
  const rotation = (Math.random() - 0.5) * 720
  return {
    transform: `translate(${x}px, ${y}px) rotate(${rotation}deg)`,
    opacity: 0,
    transition: `all 0.6s ease-in ${index * 0.05}s`,  // stagger each tile
  }
}
```

Apply via `style={scatterStyle(idx)}` on each TileCard's wrapper div during elimination.
