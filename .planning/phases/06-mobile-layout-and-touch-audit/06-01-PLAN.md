**Step 1 — Add CSS utility classes in `src/index.css`** (per research recommendation to avoid Tailwind arbitrary value parser edge cases with nested `calc(env(...))` expressions):

```css
/* Safe-area inset utilities */
.pt-safe {
  padding-top: max(env(safe-area-inset-top), 0.5rem);
}
.pb-safe {
  padding-bottom: max(env(safe-area-inset-bottom), 0.5rem);
}
```

Place these after the existing `html:has(.dark)` block. Using `max()` ensures at least 0.5rem padding on non-notched devices — never zero.
