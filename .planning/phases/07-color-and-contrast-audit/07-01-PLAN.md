Add dark mode overrides inside the existing `.dark` block:

```css

  /* existing tokens unchanged */
  --color-ink-secondary: #a09e9a;
  --color-accent-primary: #5b8de0;
  --color-accent-danger: #f07070;
}
```

Add the gradient-tension utility class AFTER the `.dark` block and before the keyframes:

```css
.gradient-tension {
  background: linear-gradient(
    to top,
    var(--color-corbusier-blue),
    var(--color-corbusier-yellow),
    var(--color-corbusier-red)
  );
}
```

**Token value rationale (from research):**
- `ink-secondary` light (#6b6967): ~4.6:1 on surface (#f2f0eb), ~4.7:1 on card (#ffffff)
- `ink-secondary` dark (#a09e9a): ~4.5:1 on surface (#1c1c24), needs verification on card (#2a2a34)
- `accent-primary` dark (#5b8de0): ~4.6:1 on dark surface — lighter blue for dark mode
- `accent-danger` dark (#f07070): ~5.0:1 on dark surface — lighter red for dark mode
